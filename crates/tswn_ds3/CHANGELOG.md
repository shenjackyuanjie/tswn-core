# 更新日志

## [0.1.0] - unreleased

### 新增

- 新增 `tswn_ds3` CLI 工具，用于基于机器学习模型执行命名评分、配对、合并、去重和排序流程。
- 支持五种单字评分模式：`bc`、`fz`、`wc`、`fs`、`pj`。
- 支持三种字对配对模式：`fc`、`wc`、`rh`。
- 提供 `run`、`show-config`、`merge`、`dedup`、`sort`、`score`、`pair` 子命令，方便拆分执行完整管道中的各阶段。
- 评分与配对路径支持线程数配置，便于在大规模输入上并行处理。

### 说明

- 该 crate 当前作为内部工具加入 workspace，版本仍处于 `0.1.0` 初始阶段。
